  • linux 在文档中查找关键字个数
    grep -o “关键字” 文档名 | wc -l
    grep -o “关键字” 文档名 | sort | uniq -c

  • 清除history记录
    vim .bash_history
    命令模式下（Esc之后输入:） 输入 set nu 每行数据前面显示行号
    11,20d 回车 11～20行的记录就被删除了
    然后命令模式下 wq 保存退出就可以了
    如果在此查看还是有记录，可以退出当前回话之后，再进去查看，就会不再显示删除的记录了
